# Lab 1: Linux, Vim and Git

## 1. BASIC LINUX COMMANDS

Commands used are:

- `echo`
- `pwd`
- `cd`
- `ls` 
- `mkdir`
- `touch`
- `rm`
- `rmdir` and others

```bash
cd
echo "HELLO, LINUX"
pwd
ls
cd ..
cd -
mkdir manisa
cd manisa

mkdir dir1 dir2 dir3
ls

touch file1.txt
touch file2.txt
ls

rm file1.txt
rmdir dir3

pwd
```
